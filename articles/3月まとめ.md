3月の朝活ブログ部は内容が非常に充実していました。データ分析の深掘り、案件探しの方法が11個に拡大、KW選定の実践的なメモ、そしてMicrosoft Clarityの導入と活用法まで。特にClarityの使い方は、ブログ改善に直結する学びでした。

## 1週目：前月データの分析＆Clarityの導入

### ブログの成長を数字で確認

メンバーのブログデータ（1月→2月の比較）を見ながら、成長ポイントと課題を分析しました。

- **記事数**：16記事（1月）→ 28記事（2月）で12記事増加
- **作業内訳**：新記事12本、リライト2本
- **ジャンル**：YMYL（健康・美容系）

2ヶ月で記事数がほぼ倍になったことは大きな進歩です。

### アフィリエイトデータの見方

クリック数が多い案件からリライトやKW再選定をすると効果的。「ゴールの手前まで来てくれているユーザー」にアプローチする考え方です。

**改善の具体的な方向性：**
- インプレッションが多いのにクリックが少ない → リンクの配置場所、リンクの個数、ボタンの色、リンクの文言、マイクロコピーの見直し

### サーチコンソール1月→2月の比較分析

1月と2月のデータを並べて比較することで、どのキーワードが伸びているか、どこが落ちているかを可視化しました。

**見るべきポイント：**
- 表示回数が多くてクリック数が少ない → タイトルを変える
- 表示回数が少ない → 検索上位を目指して表示回数を増やす
- **掲載順位10〜20位のものからリライト対象にするのがコスパ良い**
- 掲載順位50位以降はリライトしても難易度が高いので後回し
- 狙っていないキーワードからの流入があれば、そのクエリに寄せてリライトもアリ

### Microsoft Clarityの導入と活用

3月の目玉コンテンツです。Microsoft Clarityは、ユーザーの行動を可視化できる無料の分析ツール。

**WordPressへの導入手順：**
1. プラグイン「Microsoft Clarity」をインストール＆有効化
2. Clarityの設定画面からGoogleアカウントでログイン
3. プロジェクトを作成 or 接続
4. clarity.microsoft.com でデータ受信を確認

**Clarityで見るべき4つのポイント：**

| 画面 | 何がわかるか |
|------|------------|
| Dashboard | サイト全体の概要（セッション数、滞在時間、スクロール率） |
| Recordings | ユーザーの行動録画（離脱ポイントがわかる） |
| Heatmaps | クリック位置、スクロール到達点 |
| Insights | UX問題の検出（Rage Click、Dead Clickなど） |

**具体的な改善アクション：**
- スクロール率が低い → 導入文を改善、結論を先に書く、比較表を上に配置
- 比較表のクリックが少ない → CTAボタン追加、ボタンの色を変更
- Rage Click（連打クリック）が多い → クリックしても反応しない場所にリンクを設置
- すぐ離脱する → 見出し改善、ファーストビュー改善、ページ速度改善

## 3週目：キーワード選定＆競合調査（実践メモ）

### KW選定のツール一覧

ラッコキーワード、キーワードプランナー、ウーバーサジェスト、Keyword Surfer、Googleトレンド、販促カレンダー、被リンクチェックツールなど。

ノウハウ図書館のリンクも複数共有され、各ツールの具体的な使い方を学べる環境が整っています。

### KW選定の実践的なメモ（15のポイント）

交流会で出た、すぐに使えるKW選定のコツをまとめます。

1. **サジェストが出ればある程度のボリュームあり**
2. **検索して上位に個人サイトがあれば見込みあり**
3. noteがあっても上がるのは微妙かも
4. **比較や違いなどのKWは狙いやすい**
5. **新商品は出たてでライバルが少ないので狙いやすい**
6. サジェストが多い商品はロングテールKWで狙う
7. 上位に検索意図に合っていない記事が多ければチャンスあり
8. 1記事で難しければ複数記事で上位を狙う
9. ライバルサイトの記事数をチェック（同数以上が必要と仮定）
10. **allintitle:チェックでタイトルにKWが入った記事数を確認**
11. 大きなメディアは全KWを網羅しがち → 個人は細かいKWで内部リンクを繋ぐ
12. 書けるKWから書くと良い（難しいKWは後回しでOK）
13. 再検索KWを活用して新しいKWを見つける
14. **上位表示が難しい状況で記事を書くとサイト全体の評価が下がるリスクあり**
15. LPを見てKWを見つける方法もある

特に14番は重要です。勝てない記事を量産するとサイト全体の評価に影響する可能性があるため、KW選定の段階で慎重に判断することが大切です。
